🧮 Calculator in C
📌 Overview
This project implements a basic calculator using the C programming language.
It performs arithmetic operations such as addition, subtraction, multiplication, and division based on user input.

⚙️ Features
Menu-driven interface.

Supports +, -, *, / operations.

Handles both integer and floating-point numbers.

Simple and portable code (works on most compilers).


🛠️ Requirements
C compiler (e.g., Vs code, Turbo C).

Basic knowledge of C programming.

Operating system: Windows/Linux.
